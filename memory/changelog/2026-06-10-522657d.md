# 升级 GitHub Actions:checkout v6、setup-dotnet v5(Node 24)

## 做了什么
- `.github/workflows/ci.yml` 与 `.github/workflows/publish.yml` 中:
  - `actions/checkout@v4` → `@v6`(最新 v6.0.3）。
  - `actions/setup-dotnet@v4` → `@v5`(最新 v5.3.0）。

## 为什么
- 旧版本运行在 Node.js 20 上,GitHub Actions 在 2026-06-16 起强制 Node 24、
  2026-09-16 移除 Node 20,CI 日志已出现弃用警告。新主版本运行于 Node 24,清除警告。
- 纯运行环境升级,无行为变化。

## 影响的文件 / 范围
- `.github/workflows/ci.yml`
- `.github/workflows/publish.yml`

## 注意事项 / 后续
- 按主版本 tag 固定(`@v6` / `@v5`),与原 `@v4` 的固定风格一致。
- 下次 CI / 发布运行会用新版本,Trusted Publishing 的 `NuGet/login@v1` 不受影响。
