# 服务器介绍页 + 实用工具网页扩展 + AdminTools 新插件 + 全栈网页 i18n

## 做了什么
- **玩家面板首页/介绍页**:WebPanel 新增 `IntroPlayerMenu`(home tab)——管理员可在面板里编辑的
  **共享 Markdown 简介**(`IntroStore` + `WebFieldType.TextArea`)+ **按权限过滤的「可用指令列表」**
  (新增 `IPlayerCommandRegistry`/`PlayerCommandInfo`,各功能插件登记自己的玩家指令)。
- **Essentials 玩家菜单**由「便民」改名「实用工具」并扩展:`/tp` 请求的发送 + 接受/拒绝、组队
  邀请/接受/拒绝/退出/查看成员、礼包领取、睡觉投票——全部复用既有服务;为此给
  `TeleportRequestManager`/`PartyInviteManager` 加了 `PendingSenders()` 枚举。
- **player.html**:加「手动刷新 ↻ + 不会自动刷新」提示;新增 Markdown 渲染。
- **新插件 `well404.AdminTools`**(无敌/踢出/临时封禁/解封、权限组分配、按权限组授予指令):
  命令 `/god /kick /ban /unban` + 管理面板模块「管理员工具」。`GodModeService` +
  `IEventListener<UnturnedPlayerDamagingEvent>` 取消伤害;封禁走 `IUserManager.BanAsync`,解封走
  SDG `SteamBlacklist.unban`;角色走 `IPermissionRoleStore`/`IPermissionStore`/`ICommandStore`。
  **服务依赖改为按需从 LifetimeScope 解析**——否则 `IPermissionStore` 在插件作用域不可解析会
  导致整插件加载失败;改懒解析后插件正常加载,个别不可解析的能力运行时优雅失败。
- **全栈网页 i18n**(管理面板 + 玩家面板),**英文默认 + 中文可切换**:
  - 新增 `IWebTranslationRegistry`(**英文源串即键** + 各插件 `WebI18n.ZhTable` 中文映射);服务端
    按客户端 `?lang=` 渲染两个面板;两个 SPA 都加语言下拉。
  - `PlayerMenuContext.Language`、`PlayerMenuView.BodyMarkdown`、`WebFieldType.TextArea`、
    `WebActionRequest.Language`;被处理器解析的开关 Select 改为 `Boolean`(值恒 true/false,显示
    按语言 Yes/No)。
  - 现有 Economy/Shop/Essentials 的玩家菜单与管理模块文案改英文 + 中文映射表。
  - **游戏内**保持英文(`translations.yaml`),即服务器可设的 i18n 接口。
- 版本:WebPanel `0.4.0`、Economy/Shop `0.4.0`、Essentials `0.3.0`、AdminTools `0.1.0`(新)。

## 为什么
- 用户要求:玩家介绍页(含可配置简介 + 按权限的可用指令)、实用工具补齐 Essentials 能上网页的
  功能、新管理员插件、以及覆盖文档/游戏内/网页的多语言(至少英文 + 预留接口,网页玩家可自切、
  游戏内服务器端设)。无自动刷新的延迟也按要求在玩家页加了提示。
