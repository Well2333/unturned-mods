# WebPanel 集合布局（网格/列表）+ 胶囊徽章

## 做了什么
- `WebActionKind.Collection` 新增展示能力：
  - `WebPanelAction.Layout`：`"list"`=纵向列表（按实体数据，如玩家余额），默认=小方块网格
    （目录型数据，如商店商品）。
  - `WebRecord.Tags`：槽位内的多个元素各用一个**胶囊（pill）徽章**渲染、彼此间隔，
    取代原先塞进一串括号的 `名称（A，B，C）`。`Label` 只放标题。
- 宿主元数据增 `layout`、records 增 `tags`；SPA 据此渲染 `.tiles`/`.list` + `.pill`。
- Shop 商品：网格小方块，礼包内容每件一个胶囊（`名称(ID)*数量`）。
- Economy 余额：纵向列表，余额作为一个胶囊。

## 为什么
- 用户反馈：商品用小方块、玩家余额用纵向列表更直观；单槽位多元素用胶囊间隔
  比一堆括号明显更易辨认。

## 影响的文件 / 范围
- `UnturnedMods.Shared/WebPanel`（WebRecord.Tags、WebPanelAction.Layout）、
  `well404.WebPanel`（HttpServer 元数据/records、index.html 渲染+样式）、
  Economy/Shop 的 `*WebPanelModule`（记录的 Label/Tags/Layout）。

## 注意事项 / 后续
- 后端在 `.localserver` 自测通过（layout=list/grid、records tags 正确）；网格/列表/胶囊
  的视觉渲染由用户浏览器确认。
- 版本：WebPanel 0.1.1、Economy 0.2.1、Shop 0.2.1，随后按插件发布。

## 规范同步（guideline_changed = true）
