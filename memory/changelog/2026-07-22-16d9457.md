# 全仓安全审核、交互地图与可恢复交易/仓库发布

## 发布版本

- `well404.UnturnedMods.Shared` 1.4.0
- `well404.AdminTools` 1.1.1
- `well404.AutoSave` 1.1.1
- `well404.Economy` 3.0.0
- `well404.Essentials` 1.4.0
- `well404.Shop` 2.0.0
- `well404.Vault` 2.5.0
- `well404.WebPanel` 1.6.0

## 主要变更

- Essentials 增加 GPS/Chart/列表三视图互动传送地图、地图归属和边界过滤、多标签与 Emoji、地图内 Home/死亡点标记、固定尺寸叠加标记、二次确认，以及服务器持久化的紧凑/宽大偏好。
- 修复地图 UI 重渲染时旧实例销毁回调取消新实例刷新暂停的竞态；地图交互期间暂停轮询，切回列表或其他页面恢复。
- Vault 增加个人与小队共享仓库、容量购买、SQLite 状态机、CAS、审计和人工恢复界面；模糊提交不自动重放，避免复制或资产损失。
- Economy 的 SQLite 转账与账本原子化并支持幂等操作；经验值后端对跨系统交易采用失败关闭；配置更新加入验证与写时复制。
- Shop 买卖统一进入持久 saga，加入跨实例文件锁、隔离区、CAS、审计和管理员恢复；重复物品 ID 配置只采用首项，避免快速出售重复计价。
- AutoSave 将空服备份改为独立限频状态机，玩家回归恢复正常周期；升级归档依赖并用独立 TAR 读取验证输出。
- WebPanel 加固会话、语言/YAML、Markdown、请求体/超时/并发、资源大小和安全响应头；移除不安全的 cloudflared 自动下载链路与宽松 CORS。
- 发布工作流固定 Actions 提交、要求 OIDC、校验允许发布的包/版本、精确 tag 和 `origin/main` 提交，发布脚本执行同等保护。
- 清理研究中间 JSON、WIP changelog、旧工作树以及可重建的构建/浏览器测试产物；最终研究报告和通用文档保留在 `docs/`。

## 安全与验证

- 多代理分别复核安全边界、持久状态机、Shop 并发与 UI/设计；发现的跨实例交易竞态、重复出售计价、重复隔离区 DOM 和地图刷新竞态均在发布前修复并复审通过。
- `dotnet test UnturnedMods.sln --configuration Release --no-restore`：267/267 通过。
- NuGet 直接及传递依赖漏洞扫描：0 个已知漏洞；固定 `System.Formats.Asn1` 8.0.1、`System.Text.Json` 8.0.6，升级 `SharpCompress` 0.49.1。
- 全部 8 份 JavaScript 资源通过 `node --check`；shell、workflow、差异空白和敏感信息扫描通过。
- 本地专服冷启动验证 8 个候选组件各加载一次，无插件异常；测试结束后关服并确认 U3DS、cloudflared、Chrome 与测试端口均退出。
- 真实 Chrome 覆盖管理员 Essentials/Shop/Vault，以及玩家地图桌面/移动端：紧凑 648px、宽大 1124px、移动端 334px；缩放叠加、边界过滤、Home/死亡点、确认弹窗、刷新暂停和无横向溢出均通过。

## 边界

- 玩家原生 GPS/Chart 资格、实体传送落点以及非普通 Cartography Volume 的最终视觉对照仍需在线玩家环境；自动化与本地浏览器验证未放宽任何在线或权限校验。
- Vault/Shop 跨 Unity 与 SQLite 无法组成单一 ACID 事务；不确定状态保留完整审计并要求管理员明确处理，不进行可能复制资产的自动猜测。
